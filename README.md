## Guides (standalone docs)

Each doc is a standalone guide with links and references. Start with the [checklist](docs/checklist.md) to tie everything together.

| Doc | What it covers |
|-----|----------------|
| **[Monitors](docs/monitors.md)** | Resolution (1080p/1440p/4K), refresh rate, panel types (IPS/VA/OLED), VRR (G-Sync/FreeSync), size/distance, sharing one set between Mac + PC. Links: RTINGS, Wirecutter, r/Monitors. |
| **[Desk and switching](docs/desk-and-switching.md)** | KVM vs Barrier/Synergy vs monitor input only; dual-monitor KVM options; cable management. Links: Barrier, Synergy, KVM vendors. |
| **[PC specs](docs/pc-specs.md)** | GPU tiers by resolution, CPU/RAM/storage, prebuilt vs build-your-own, budget tiers and timing. Links: r/buildapc, PC Part Picker, GPU hierarchy. |
| **[Peripherals](docs/peripherals.md)** | Keyboard, mouse, headset — shared vs dedicated; Mac/PC compatibility; USB vs Bluetooth vs dongle. |
| **[Checklist and references](docs/checklist.md)** | Step-by-step checklist (monitor → switching → PC → peripherals → buying) and consolidated reference links. |
